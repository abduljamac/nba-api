# NBA API

This API was built with data scraped from espn.com/nba

### Built With:
- Node
- Express
- Firebase
- Cheerio/Request

### Quick Start
1. Clone the repo
2. Build a docker image: `docker build -t nba-api .`
3. Run a docker container: `docker run -p 8000:8000 nba-api`
4. Go to `http://localhost:8000`
